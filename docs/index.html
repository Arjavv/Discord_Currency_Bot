Slash Commands

A comprehensive list of bot commands to interact with the Soul Currency system.

/checkin User
Claim your daily allowance reward.

Triggers the 24-hour daily allowance checkin. Awards 20 Souls atomically. If run while on cooldown, displays a relative unlock timer embed.

Syntax: /checkin
Channel restriction: Runs ONLY in #soul-bot (ephemeral block in other channels).
/balance [user] User
Check your wallet balance or search for another user's balance.

Fetches the balance of Souls. You can optionally check another user's wallet balance by tagging them in the parameter.

Syntax: /balance [user: @username]
Channel restriction: Runs ONLY in #soul-bot.
/leaderboard User
View the top 10 richest users in the current cycle.

Displays an embed listing the top 10 richest users ranked in descending order of their current active Soul balances.

Syntax: /leaderboard
Channel restriction: Runs ONLY in #soul-bot.
/casino flip <choice> <bet> Casino
Flip a coin for double or nothing.

Bets a specific positive amount of Souls on a coin toss outcome. Correct guesses award twice the bet amount (wins net your bet amount). Wrong guesses deduct the bet amount. Checks for balance sufficiency before running.

Syntax: /casino flip choice:[Heads/Tails] bet:[positive integer]
Channel restriction: Runs ONLY in #soul-bot.
/admin <subcommand> Admin
Administrative controls for settings, setup, and cycle resets.

Set server-wide currency options, initialize channels, or reset cycles. Restricted to Server Administrators only. The bot validates administrator permissions both inside Discord interface rules and secondary runtime checks on execution.

  • setup: Automatically creates the category "Soul" and the #soul-bot and #admin-logs channels if they do not exist.
  • set-currency-name <name>: Customizes the display name of the currency server-wide (default is Souls).
  • set-currency-icon <icon>: Sets a new custom emoji tag as the balance symbol.
  • reset-cycle: Closes current monthly cycle, archives the rankings, and resets all balances & message counters to 0 server-wide.
Permission: Server Administrators only.

Cycle Archival System

Manual Monthly Reset Wipes

To keep server standings competitive and fresh, currency balances and chat counters do not persist forever. Administrators manually archive standings into snapshots at the end of each month before wiping active balances to 0 for a new cycle.
